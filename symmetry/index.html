Chapter 1: Symmetry

Group theory is about symmetry. For example, a square is very symmetrical.

But what does that mean? Well, it means we can do some action to it and it'd still look identical.

We can rotate it, flip it, etc. we call each one of these actions a symmetry and if we were to list out each and every one of these symmetries, we get a group.

The central idea of group theory isn't just listing out every single symmetry and then calling it a day. One important aspect of it is seeing how symmetries interact with other symmetries.
